Class com.xerox.integrityperformancetool.IntegrityMetricBean

java.lang.Object

A java bean abstracting a single metric retrieved from a Server Metrics .txt file.

NOTE: The Server Metrics file can be generated from an Integrity Server via the Integrity Admin Client or the CLI command "im diag diag=metrics > serverMetrics.txt".

An IntegrityMetricBean object has the following private properties:

  • customName (java.lang.String) - The custom name assigned to the IntegrityMetricBean, which may be different than the actual metric name given in the Server Metrics .txt file. This property may be used to assign a more useful name to a given IntegrityMetricBean than the actual name provided in the Server Metrics .txt file. The default value is "None".
  • metricName (java.lang.String) - The name of the metric as retrieved from the Server Metrics .txt file. The default value is "None".
  • value (java.lang.Object) - The value of the metric as retrieved from the Server Metrics .txt file. There is no default value.
